# AGENTS.md (repo router)

Genuine Gossip is a polyglot single repo.

- Mobile app (Flutter) → `apps/mobile/AGENTS.md`
- Marketing (Astro) → `apps/marketing/AGENTS.md` (not yet created)
- Admin → `apps/admin/AGENTS.md` (not yet created)
- Backend → Supabase, see `supabase/` (not yet created)
- Design tokens / brand / writing → `design/` (not yet created)

Determine which app the working file lives under and read that app's AGENTS.md.
Repo-wide rules (when they exist) live in root `.agents/`.
