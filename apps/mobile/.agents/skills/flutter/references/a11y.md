# Accessibility & readability

Hard, project-agnostic defaults. Apply whenever choosing colors, type sizes, or text layout.

## Color contrast (WCAG 2.1 AA)

- Normal text: contrast ratio ≥ **4.5:1** against its background.
- Large text (≥ 18pt regular or ≥ 14pt bold): contrast ratio ≥ **3:1**.

Verify any non-trivial color pair against these before shipping.

## Typography readability

- **Line height (leading):** **1.4×–1.6×** the font size for body text.
- **Line length:** aim for **45–75 characters** per line for body copy.
- **Avoid all-caps for long-form text.** Reserve all-caps for short labels.

## Flutter hooks

- Use the `Semantics` widget to attach descriptive labels to interactive or
  non-textual UI elements.
- Test with TalkBack (Android) and VoiceOver (iOS) for any screen that ships
  to users.
- Verify the UI remains usable when system text scaling is increased.
